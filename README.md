# Лабораторная работа №2
Вам необходимо доработать написанный вами в рамках лаб. работы №1 калькулятор.

Что необходимо сделать:
1.	Каждую математическую операцию вынести в отдельный метод;
2.	Добавить к существующему функционалу операцию получения остатка от деления большего числа на меньшее;
3.	Если оба числа введенных числа четные, то рассчитать для них квадрат суммы, иначе - квадрат разности.
4.	Добавить меню: считать не все математические операции, а только ту, которую выбрал пользователь.
