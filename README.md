# Лабораторная работа №1
Вам нужно разработать калькулятор

Что необходимо сделать:
1.	Осуществить ввод двух чисел
2.	Для введенных пользователем чисел считаются след. операции: 
a.	умножение
b.	деление
c.	сумма
d.	разность
3.	Рассчитанные результаты операций выводятся на экран
4.	Калькулятор продолжает работать до тех пор, пока пользователь не нажмет кнопку, которая завершит работу калькулятора.
5.	Перед выводом расчетов необходимо вывести введенные пользователем числа.
