# Лабораторная работа №3

Вам необходимо разработать программу, которая автоматизирует работу логистической компании.
Что необходимо сделать:
1.	В вашей программе должны быть след. сущности:
	- логист
		- автомобиль
		- водитель
		- маршрутный лист
		- груз

2.	Логист должен содержать следующие характеристики:
	- имя
	- возраст
	- список подконтрольных ему водителей

3.	Логист должен уметь: 
	- отправить водителя в конкретную точку его маршрутного листа
	- запросить у водителя его текущее местонахождение
	- дать указание водителю разгрузиться в текущем местонахождении
	- назначит любого свободного водителя на новый маршрут
	- заполнить маршрутный лист
	- Дать указание водителю загрузить машину (дать перечень грузов)

4.	Водитель должен иметь следующие характеристики:
	-	Имя
	-	Возраст
	-	Водительский опыт (в годах)
	-	Машина, на которой он перевозит грузы
	-	Маршрутный лист

5.	Водитель должен уметь
	-	Доехать на автомобиле до указанной точки из маршрутного листа
	-	Заправить автомобиль
	-	Разгрузить машину
	-	Загрузить машину
	-	Остановиться на ночевку, если расстояние, на которое его отправляют, больше 300км
	-	Проинформировать логиста о текущем местоположении

6.	Автомобиль должен иметь след. характеристики:	
	-	Максимальная грузоподъемность
	-	Марка
	-	Год производства
	-	Объем топливного бака
	-	Расход топлива на 100 км

7.	Автомобиль должен уметь:
	-	Доезжать до указанной точки
	-	Заправляться
	-	Двигаться
	-	Останавливаться
	-	Загружаться
	-	Разгружаться

8.	Должно существовать 3 разновидности автомобиля:
    -   Малый (грузоподъемность 100 кг)
	-	Средний (грузоподъемность 350 кг)
	-	Большой (грузоподъемность 1000 кг)

9.	Груз должен обладать след. характеристиками:
-	Вес
-	Наименование
